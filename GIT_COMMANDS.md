# Upload to GitHub

Create a GitHub repository named:

```text
proteuslab-autodraw-orchestrator
```

Then run:

```bash
git init
git add .
git commit -m "Release PLADO v0.6.2 identity and whitepaper"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/proteuslab-autodraw-orchestrator.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.
