# FAQ

## Can it save in Proteus Demo?

No. It presses the real Save command. If Demo blocks saving, Proteus refuses.

## Can it generate `.pdsprj` files without Proteus?

No. Real projects must be saved by Proteus.

## Why calibration?

Every screen, zoom level, and Proteus layout is different.

## Does it work for everyone?

It is designed for public use, but each user must calibrate first.
