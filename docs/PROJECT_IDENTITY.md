## One-Sentence Description

PLADO is a calibration-aware GUI automation framework that transforms digital-circuit lab task templates into guided Proteus ISIS schematic construction workflows with verification screenshots and submission packaging.

## Extended Description

ProteusLab AutoDraw Orchestrator provides a structured automation layer over Proteus ISIS for educational digital electronics laboratories. Instead of attempting to reverse-engineer proprietary project formats, PLADO uses calibrated GUI interaction. Each user records a local profile containing the locations of important Proteus interface elements and schematic reference points. Circuit templates then map component placements and wire paths to that calibrated coordinate system.

This makes the framework portable across different machines while preserving a transparent and legal workflow. PLADO acts as a precision assistant for repetitive lab construction, not as a replacement for Proteus and not as a license bypass mechanism.
