# Simplified Calibration

Starting from v0.6.3, calibration is simpler.

The tool no longer asks for the Pick Devices `P` button.

## New method

1. Open Proteus.
2. Open Pick Devices manually.
3. Run:

```bash
proteus-lab calibrate
```

4. Capture:
   - search box;
   - OK button;
   - schematic origin.

During AutoDraw, PLADO opens Pick Devices using the `P` keyboard shortcut instead of a calibrated button click.
